Connection Refused

Causes of this could be:

  • The port isn't open
  • The website isn't online
  • The server has you as an blacklisted IP address (Very unlikely)
  • The server is so busy helping everyone else, you got lost in the crowd. (The server is being overworked)
  • Firewalls

What to do:

  • Open up the firewalls
  • Wait and try again later
  • If you're really determined, try to find the owner of the websites email (through ICANN Lookup*)

If you want to be more knowledgeable on why the website won't accept you, check out this Stack Overflow answer

*Sometimes, a website is protected by "Whois Privacy Protection Service, Inc." This is when the website owner does not want to be looked up, or their information not to be made public. Under these circumstances, all you can do is just wait patiently.
